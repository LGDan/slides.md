## Getting Started

```bash
# Build the project
go build -o slides main.go

# Run with defaults
./slides

# Customize
./slides -file=my-slides.md -theme=dracula -port=3000
```

## Navigation

- **Arrow Right** or **Space**: Next slide
- **Arrow Left**: Previous slide
- **Buttons**: Click to navigate
